# Create a calculator using functions

create the 4 operator functions
```
def add(num1, num2):
    return num1 + num2


def subtract(num1, num2):
    return num1 - num2


def multiply(num1, num2):
    return num1 * num2


def divide(num1, num2):
    return num1 / num2
```
print the functions and pass the values to each function variable
```
print(add(2, 2))
print(subtract(2, 2))
print(multiply(2, 2))
print(divide(2, 2))
```
